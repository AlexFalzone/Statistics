Probability Space and Examples

Sample Space (Ω)

The sample space is the set of all possible outcomes of a random experiment. It is denoted by Ω (omega). For example, if you roll a six-sided die, the sample space would be {1, 2, 3, 4, 5, 6}.

Event Space (F)

The event space is a collection of subsets of the sample space, representing possible events or combinations of outcomes. It is denoted by F. For the six-sided die example, an event might be "rolling an even number," and its corresponding subset could be {2, 4, 6}.

Probability Measure (P)

The probability measure assigns a probability to each event in the event space, satisfying certain axioms. It is denoted by P. The probability of an event A is written as P(A). The probabilities must satisfy the following conditions:

  • P(A) is non-negative for every event A.
  • P(S) = 1, where S is the entire sample space.
  • For any sequence of mutually exclusive events A1, A2, ..., the probability of the union of these events is the sum of their individual probabilities: P(A1 ∪ A2 ∪ ...) = P(A1) + P(A2) + ...

Example: Tossing a Fair Coin

Sample Space (Ω): {H, T} (Heads, Tails)

Event Space (F): {{}, {H}, {T}, {H, T}} (empty set, heads, tails, both)

Probability Measure (P): P({}) = 0, P({H}) = 0.5, P({T}) = 0.5, P({H, T}) = 1

Exercise 1: Rolling a Fair Six-sided Die and Getting an Even Number

Sample Space (Ω): {1, 2, 3, 4, 5, 6}

Event Space (F): {{}, {2, 4, 6}, {1, 3, 5}, {2}, {4}, {6}} (empty set, even numbers, odd numbers, rolling a 2, rolling a 4, rolling a 6)

Probability Measure (P): P({}) = 0, P({2, 4, 6}) = 0.5, P({1, 3, 5}) = 0.5, P({2}) = P({4}) = P({6}) = 1/6
